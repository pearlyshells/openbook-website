Openbook Health logo kit
=========================

Source basis:
This kit is derived from the existing inline logo mark found in the supplied website file.

Included files:
- openbook-mark-gold.svg
- openbook-mark-white.svg
- openbook-mark-navy.svg
- openbook-lockup-horizontal-dark.svg
- openbook-lockup-horizontal-light.svg
- openbook-lockup-stacked-dark.svg
- favicon.svg

Suggested brand colors:
- Navy: #080b12
- Mid Navy: #0e1422
- Gold: #c9963a
- Gold Light: #e4b65a
- Off White: #f4f1ec

Usage guidance:
- Use mark-gold or lockup-horizontal-dark on dark backgrounds
- Use mark-navy or lockup-horizontal-light on light backgrounds
- Use favicon.svg for browser tab icon
- Keep generous clear space around the mark; avoid stretching or changing stroke relationships
